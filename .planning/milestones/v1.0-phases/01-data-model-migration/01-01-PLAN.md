## Goal

Define all door hardware data structures, seed catalog, default hardware sets, and the isDoorSystemType() utility. After this plan, the app has typed interfaces for door hardware, 12 seed items with pricing, 4 default hardware sets (one per door system type), and a tested utility that identifies door systems.

## Context for Executor

**Codebase patterns to follow:**
- Seed data arrays: see `src/data/seed-hardware.ts` -- typed arrays exported from `src/data/seed-*.ts`
- Types: all interfaces in `src/types/index.ts`
- Calc utilities: pure functions in `src/calc/` with barrel export in `src/calc/index.ts`
- Hardware IDs use `hw-xxx` prefix; door hardware uses `dhw-xxx` prefix
- Tests live alongside source as `*.test.ts` with inline factory helpers

**User decisions (LOCKED -- do not change):**
- sys-006 (Entrance System) IS a door type -- 4 types total, not 3
- Swing Door (sys-009): Hinges (3), Closer (1), Handle/Pull (1), Lock/Cylinder (1), Threshold (1), Weatherstrip (1)
- Sliding Door (sys-008): Handle/Pull (2), Lock/Cylinder (1), Threshold (1), Weatherstrip (1)
- Revolving Door (sys-007): Handle/Pull (2), Lock/Cylinder (1), Threshold (1), Weatherstrip (1), Sweep (1), Auto-Operator (1)
- Entrance System (sys-006): Hinges (3), Closer (1), Handle/Pull (2), Lock/Cylinder (1), Threshold (1), Weatherstrip (1), Pivot (2)
- Door hardware is SEPARATE from glazing hardware -- own section in Settings, own seed file
- Default hinge quantity is static at 3 (smart suggestion is Phase 2)

```typescript
/**
 * Door system type identification utility (CALC-02).
 * Single source of truth for which system types are door types.
 */
export const DOOR_SYSTEM_IDS: ReadonlySet<string> = new Set([
  'sys-006',  // Entrance System
  'sys-007',  // Revolving Door
  'sys-008',  // Sliding Door
  'sys-009',  // Swing Door
])

/**
 * Returns true if the given system type ID is a door system type.
 * Used by calc engine (Phase 2) and hooks (Phase 3) to apply
 * door-specific logic.
 */
export function isDoorSystemType(systemTypeId: string): boolean {
  return DOOR_SYSTEM_IDS.has(systemTypeId)
}
```
